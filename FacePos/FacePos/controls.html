Buttons

The colors used for the background and text of buttons have been vetted to meet a minimum color contrast ratio of 4.5:1 for accesibility purposes.

Using <a> tag. The <a> tag should be used for links between pages, not for JavaScript actions.

Rest state

Default Primary Success Info Warning Danger

Disabled state

Default disabled Primary disabled Success disabled Info disabled Warning disabled Danger disabled

Active (on click/mouse down) state

Default active Primary active Success active Info active Warning active Danger active 

<a href="" class="btn [btn-default] [btn-primary] [btn-success] [btn-info] [btn-warning] [btn-danger] [disabled] [active]">
    Button link
</a>

Using <button> tag. The <button> tag is used on forms and for JavaScript actions.

Rest state

Disabled state

Active (on click/mouse down) state

<button type="button"
       [disabled]
        class="btn [btn-default] [btn-primary] [btn-success] [btn-info] [btn-warning] [btn-danger] [active]">
    Button
</button>

Buttons sizes

Button sizes from extra small, small, normal to large. 

<button type="button" class="btn btn-primary btn-xs">Extra small button</button>
<button type="button" class="btn btn-primary btn-sm">Small button</button>
<button type="button" class="btn btn-primary">Normal button</button>
<button type="button" class="btn btn-primary btn-lg">Large button</button>
<button type="button" class="btn btn-primary win-btn-xl">Extra large button</button>

A full width button that can be used with the various sizes. 

<button type="button" class="btn btn-primary btn-block btn-xs">Extra small block button</button>
<button type="button" class="btn btn-primary btn-block btn-sm">Small block button</button>
<button type="button" class="btn btn-primary btn-block">Normal block button</button>
<button type="button" class="btn btn-primary btn-block btn-lg">Large block button</button>
<button type="button" class="btn btn-primary btn-block win-btn-xl">Extra large block button</button>

Tooltips

This is a simple tooltip on top (default). On right. On the bottom. On the left.

You can also place them on buttons.

To enable the tooltip on the HTML: 

<span data-toggle="tooltip"
     [data-placement="top|right|bottom|left"]
      title="Tooltip content">
    Has a tooltip
</span>

And the required JavaScript initialization: 

$(function () {
    $('[data-toggle="tooltip"]').tooltip();
});

Example with a button to launch the modal:

<button type="button" class="btn btn-primary btn-lg"
        data-toggle="modal"
        data-target="#modal-bootstrap-sample">
    Launch demo modal
</button>

Launching the modal with the JavaScript API:

$('#modal-bootstrap-sample').modal('show');

Closing the modal with the JavaScript API:

$('#modal-bootstrap-sample').modal('hide');

By default the modal is already in the DOM:

<!-- Add win-modal-center to center the modal vertically -->

     tabindex="-1" role="dialog"
