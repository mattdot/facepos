Page header

The basic structure: 

<div class="page-header">
    <div class="container">
        <div class="row">
            <div class="col-xs-24">
                <h1>This is a page header</h1>
                <p>With some content</p>

                <!--
                    Add data-win-scroll="smooth" attribute
                    and the following JS code
                    to enable smooth scrolling
                -->
                <ul class="nav nav-pills"
                    [data-win-scroll="smooth" ]>
                    <li role="presentation" class="active">
                        <a href="#nav-item-1">Nav item 1</a>
                    </li>
                    <li role="presentation">
                        <a href="#nav-item-2">Nav item 2</a>
                    </li>
                    <li role="presentation">
                        <a href="#nav-item-3">Nav item 3</a>
                    </li>
                </ul>
            </div>
        </div>
    </div>
</div>

The JavaScript code to enable smooth scrolling: 

$(function () {
    $('[data-win-scroll="smooth"] a[href*=#]:not([href=#])').click(function () {
        if (location.pathname.replace(/^\//, '') === this.pathname.replace(/^\//, '') &&
            location.hostname === this.hostname) {

            var target = $(this.hash);
            target = target.length ? target : $('[name=' + this.hash.slice(1) + ']');

            if (target.length) {
                $('html, body').animate({
                    scrollTop: target.offset().top
                }, 1500);

                return false; // prevent default href
            }
        }
    });
});

More

This is the generic structure to display a section with a header and view all link.

<section class="win-section">
    <header class="win-section-header row">
        <h2 class="win-section-title col-xs-24">
            <a href="#">This is a header</a>
        </h2>

        <div class="win-section-header-aside">
            <a href="#" class="btn btn-link section-header-more">More</a>
        </div>
    </header>

    <div class="row">
        <div class="col-xs-24">
            <!-- Content of the section -->
        </div>
    </div>
</section>

Section gallery

This is a generic section of a gallery layout. The media containers have a bottom margin to nicely layout them.

<section class="win-section win-section-gallery">
    <header class="win-section-header">
        <h1 class="win-section-title">Section gallery</h1>
    </header>

    <ul class="list-unstyled row">
        <!-- Repeated element -->
        <li class="col-md-6">
            <figure class="win-media">
                <div class="win-media-img win-ratio-2-1">
                    <a href="#">

                    <h4 class="win-media-header">
                        <a href="#">Header</a>
                    </h4>
                    <div class="win-media-subheader">
                        <a href="#">Subheader</a>
                    </div>
                </figcaption>
            </figure>
        </li>
